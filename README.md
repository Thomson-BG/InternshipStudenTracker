# Bulldog Garage Intern Tracker

A full-stack web application for tracking interns at Bulldog Garage.

## Project Structure

- `/client` - Frontend files (HTML, CSS, JavaScript)
- `/data/seeds` - Database seed data
- `/server` - Backend API and database connection
- `package.json` - Project dependencies and scripts

## Getting Started

1. Install dependencies
```bash
npm install
```

2. Start the development server
```bash
npm run dev
```

## API

The backend provides a simple API at `/api` that returns:
```json
{
  "message": "Welcome to the Bulldog Garage Intern Tracker API!"
}
```
